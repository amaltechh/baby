## üåü Key Features

### **For Parents**
- **Milestone Tracking:** Log and visualize baby growth milestones and daily health metrics.
- **Product Catalog:** Browse and manage specialized baby products with detailed descriptions.
- **Scheduling:** Book and manage daycare appointments or feeding/sleep schedules.
- **Secure Authentication:** Private user accounts handled via JWT and Firebase.

### **For Administrators**
- **Inventory Management:** Add, update, or remove products from the platform.
- **User Management:** Monitor registrations and service requests.
- **Analytics:** View dashboard statistics on service usage and popular products.
